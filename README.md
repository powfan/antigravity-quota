# antigravity-quota

Check and visualize Antigravity model quotas in your terminal.

[中文](#中文)

## Usage

```bash
npx antigravity-quota
```

## Example Output

```

  Claude Opus 4.5 (Thinking) ━━━━━━━━━━━━━━━━━━──  91%  3h28m
  Claude Sonnet 4.5          ━━━━━━━━━━━━━━━━━━──  91%  3h28m
  Gemini 3 Pro (High)        ━━━━━━━━━━━━━━━━━━━━ 100%  4h30m
```

## Prerequisites

- Antigravity (Cursor) must be running
- macOS or Linux
